# Simon Says Game 🎮

**Simon Says** is an interactive memory game built using **HTML, CSS, and JavaScript**. It challenges players to remember and repeat an increasingly long sequence of button flashes. This version features an **animal theme**, sound effects, and mobile-friendly design, making it engaging on both desktop and mobile devices.

---

## Features

- **Animal-Themed Buttons:** Leopard, Tiger, Lion, and Cheetah with unique images.  
- **Flash Animations:** Buttons flash during the game sequence, and user clicks are highlighted.  
- **Sound Effects:**  
  - Correct click sound for successful steps  
  - Error sound for mistakes  
- **Wrong Button Red Flash:** Only the incorrect button flashes red for immediate visual feedback.  
- **Level & Score Display:** Current level shown, and score displayed on game over.  
- **Popup Instructions:** Easy onboarding for new players.  
- **Mobile-Friendly UI:** Responsive buttons and text for smaller screens.  
- **Professional Design:** Clean layout, portfolio-ready, works on desktop and mobile.

---

## How to Play

1. Press any key (desktop) or tap anywhere (mobile) to start the game.  
2. Watch the sequence of button flashes carefully.  
3. Repeat the sequence by clicking/tapping the buttons in the same order.  
4. The sequence grows longer with each level, challenging your memory.  
5. If a mistake is made, the wrong button flashes red, a sound plays, and the game ends.  
6. Press any key or tap to restart and try to beat your previous score.

---

## Technologies Used

- **HTML** – structure of the game and popup  
- **CSS** – styling, button animations, overlay effects, responsive design  
- **JavaScript** – game logic, sequence generation, sound effects, flash animations
